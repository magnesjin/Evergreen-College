# Modern College Management System

A comprehensive, modern web application for college management, built with React and Vite. It provides an intuitive, high-performance interface with top-tier design aesthetics, responsive layouts, and seamless navigation.

## Features
- **Comprehensive Navigation**: Includes mega-menus for easy access to various sections.
- **Dynamic Pages**: Dedicated sections for About, Academics, Administration, Research, Campus Life, Admissions, and Contact.
- **Admission System**: Features a functional PDF admission form download.
- **Premium Design Aesthetics**: Includes modern typography (Inter), glassmorphism effects, smooth animations, and a sleek dark mode layout.
- **Responsive Layouts**: Fully responsive across mobile, tablet, and desktop viewports.

## Tech Stack
- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS with modern custom properties and flexbox/grid layouts.
- **Icons**: React Icons (Lucide/Feather styles).
- **Routing**: React Router DOM.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd "College Management"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
To build the application for production, run:
```bash
npm run build
```
This will generate optimized static files in the `dist` directory.
